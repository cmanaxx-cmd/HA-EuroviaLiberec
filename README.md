# HA Eurovia Liberec - Modbus PLC Integration

Custom Home Assistant integration pro čtení dat z PLC přes Modbus TCP s ukládáním historických teplot a elektroměrů.

## Struktura

- `custom_components/eurovia_liberec/` - Custom Integration
- `modbus_db.py` - SQLite databáze
- `export_to_excel.py` - Export do Excelu
- `configuration.yaml` - HA konfigurace
- `lovelace_dashboard.yaml` - Dashboard konfigurace

## Instalace

1. Klonuj repo:
```bash
cd /config/custom_components
git clone https://github.com/cmanaxx-cmd/HA-EuroviaLiberec.git eurovia_liberec
```

2. Restartuj Home Assistant

3. Přidej do `configuration.yaml`:
```yaml
eurovia_liberec:
  host: 192.168.1.100
  port: 502
  slave_id: 1
```

## Датасеты

### Teploty (Registry 27-34, děleno 10)
- 27: Teplota radiátor 1
- 28: Teplota radiátor 2
- 29: Teplota radiátor 3
- 30: Teplota akumulátor 1 horní
- 31: Teplota akumulátor 1 spodní
- 32: Teplota akumulátor 2 horní
- 33: Teplota akumulátor 2 spodní
- 34: Teplota topné čidlo 1/2

### Čas a Datum (Registry 35-37)
- 35: Hodiny:Minuty
- 36: Den:Měsíc
- 37: Rok

### Elektroměry a Kalorimetr (Registry 10-25)
- 10-25: Elektroměry a Kalorimetr (bez konverze)

### Synchronizace
- Register 25: Poslední uložená pozice v PLC (0-2000)
- Register 500: Pozice kterou chce vyčíst HA
- Když se 25 ≠ 500 → nová data k vyčtení

## SQLite Databáze

Data se ukládají do `/config/eurovia_db.sqlite`

**Tabulka: `measurements`**
```
id, timestamp, temp_rad1, temp_rad2, temp_rad3, temp_aku1h, temp_aku1s, temp_aku2h, temp_aku2s, temp_tc1, temp_tc2, ..., created_at
```

## Export do Excelu

```bash
python3 /config/export_to_excel.py
```

Vytvoří soubor: `/config/eurovia_export_YYYY-MM-DD.xlsx`

## Dashboard

Dashboard je dostupný v HA na kartě "Eurovia Liberec"

- Tabulka s historickými daty
- Grafy teplot
- Aktuální hodnoty

## Troubleshooting

### Integraci se nenačítá
- Zkontroluj IP adresu PLC (192.168.1.100)
- Zkontroluj port (502)
- Zkontroluj připojení k síti

### Data se neukládají
- Zkontroluj práva k `/config` adresáři
- Restartuj Home Assistant
